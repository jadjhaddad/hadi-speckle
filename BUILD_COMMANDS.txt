If you can't run scripts, copy and paste these commands one by one

----------------------------------------------------------------
STEP 1: Clean Solution
----------------------------------------------------------------
dotnet clean All.sln

----------------------------------------------------------------
STEP 2: Remove bin and obj folders
----------------------------------------------------------------
Get-ChildItem -Path . -Include bin,obj -Recurse -Directory | Remove-Item -Recurse -Force

OR if using Command Prompt:
for /d /r . %d in (bin obj) do @if exist "%d" rd /s /q "%d"

----------------------------------------------------------------
STEP 3: Clean ETABS Plug-Ins folder
----------------------------------------------------------------
Remove-Item "C:\ProgramData\Computers and Structures\ETABS 22\Plug-Ins\*.dll" -Force

OR if using Command Prompt:
del /q "C:\ProgramData\Computers and Structures\ETABS 22\Plug-Ins\*.dll"

----------------------------------------------------------------
STEP 4: Build Core
----------------------------------------------------------------
dotnet build Core\Core\Core.csproj -c Debug /p:SkipHusky=true

----------------------------------------------------------------
STEP 5: Build Objects
----------------------------------------------------------------
dotnet build Objects\Objects\Objects.csproj -c Debug /p:SkipHusky=true

----------------------------------------------------------------
STEP 6: Build DesktopUI2

dotnet build DesktopUI2\DesktopUI2\DesktopUI2.csproj -c Debug /p:SkipHusky=true

STEP 7: Build ConverterETABS22
